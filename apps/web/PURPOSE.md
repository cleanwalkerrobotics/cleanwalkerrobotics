# apps/web — Purpose

Marketing website and interactive demos for CleanWalker Robotics.

**Owner:** cw-software
**What belongs here:** Pages, components, API routes, demo applications, static assets (renders, 3D models), and web-specific configuration.
**What does NOT belong here:** Backend business logic (→ services/), shared types (→ packages/types/), documentation (→ docs/), ML models (→ ml/).
**Key constraint:** All demos must be registered in `src/app/demos/registry.json`.
